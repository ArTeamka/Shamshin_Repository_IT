# Тест-кейсы для сайта Shamshin IT

## 1. Тестирование верстки

```bash
w3c-validator index.html
```
**Ожидаемый результат**: Нет ошибок валидации

### 1.2. Адаптивность
- [ ] Проверить на мобильных устройствах (ширина 320px-768px)
- [ ] Проверить на планшетах (768px-1024px)
- [ ] Проверить на десктопах (1024px+)

## 2. Функциональное тестирование

1. Нажать кнопку "Узнать больше"

### 2.3. Навигационное меню
1. Кликнуть по каждой ссылке в меню
2. Проверить плавность прокрутки
